An Image Search Microservice By dom.farnham('@JSCodebook')

A microservice which takes your search string, searches the web for images related to the string and
then returns JSON containing the image's URLs, Alt text and the URL of the website hosting the image.

Example usage:

Search for images. Use offset parameter for pagination (optional).
https://image-search-microservice.glitch.me/api/search/funny%20cats?offset=2

View a list of recent searches.
https://image-search-microservice.glitch.me/api/recent
